# Contributing

欢迎补充新的产品经理方法论、模板和示例。

## 新增方法论

请优先补充到 `personas/product-leaders.md` 或 `personas/archetypes.md`。

要求：

- 使用公开材料。
- 写成方法论镜头，不写成真人扮演。
- 包含适合场景、关注问题、输出偏好。
- 至少能帮助生成 `Spec.md`、`PRD.md` 或 `DevDoc.md`。

## 新增模板

请放到 `templates/`。

模板应该：

- 能被直接复制使用。
- 有清晰标题和章节。
- 包含可测试的验收标准。
- 避免只写概念，不写交付。

## 新增示例

请放到 `examples/`。

示例应该：

- 面向独立开发者或 AI 编程新人。
- 范围足够小，可以作为 MVP。
- 文档完整到可以直接开工。

## 不接受

- 冒充真实人物口吻。
- 未经授权的私人材料。
- 长篇复制书籍、课程或付费内容。
- 只有鸡汤，没有产品决策价值的内容。
