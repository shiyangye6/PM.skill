# Persona Distillation Workflow

这个流程用于新增一个产品经理、产品思想家或产品团队方法论。

## 原则

- 只使用公开材料。
- 不冒充本人。
- 不收集或使用私人聊天、内部文档、泄露资料。
- 不复制长段原文。
- 把人名转化为“方法论镜头”，供产品分析使用。

## 信息来源

可接受：

- 公开书籍中的核心框架
- 公开演讲
- 公开访谈
- 本人公开文章
- 公开课程或播客摘要
- 被广泛引用的产品方法论

不接受：

- 私人聊天记录
- 公司内部资料
- 未经授权的付费内容全文
- 无法验证的二手转述

## 蒸馏模板

```md
### <姓名或方法论名称> 型：<一句话定位>

适合：

- <产品类型>

关注问题：

- <问题 1>
- <问题 2>
- <问题 3>

输出偏好：

- <产物 1>
- <产物 2>

不适合：

- <边界>
```

## 质量检查

- 是否能用于实际产品决策？
- 是否避免了模仿口吻和人格表演？
- 是否能和其他产品镜头形成互补？
- 是否能产出文档，而不只是观点？
